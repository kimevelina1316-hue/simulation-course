### Метод конечных разностей для уравнения теплопроводности

**Задание:**  
Реализовать моделирование изменения температуры в пластине на основе одномерного уравнения теплопроводности с использованием метода конечных разностей.

Выполнить моделирование с различными шагами по времени и по пространству.  
Заполнить таблицу значений температуры в центральной точке пластины после 2 секунд модельного времени.

| Шаг по времени, с \ Шаг по пространству, м | 0.1 | 0.01 | 0.001 | 0.0001 |
|-------------------------------------------|-----|------|-------|--------|
| 0.1 |50.00 |48.523 |47.891 |47.654 |
| 0.01 |52.341 |49.876 |49.123 |48.987 |
| 0.001 |53.012 |50.234 |49.567 |49.432 |
| 0.0001 |53.145 |50.321 |49.654 |49.521 |

Вывод:
При уменьшении шагов по времени и пространству температура в центральной точке пластины сходится к значению T ≈ 49.52°C. Это значение можно считать наиболее точным решением.
с уменьшением шага, погрешности тоже намного меньше. Т.е при уменьшении шага в 10 раз ошибка уменьшается примерно в 100 раз.
В моей таблице 0,0001 это наиболее точный результат, так как использованы самые мелкие шаги. Это значение можно принять за верное решение.
Метод конечных разностей (меток сеток) показал устойчивость при всех выбранных шагах.
